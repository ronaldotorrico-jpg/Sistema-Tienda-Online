Sistema de Tienda Online
========================

Segundo Parcial

Descripción
-----------
Sistema web desarrollado con Flask y Flask-AppBuilder para la gestión de una tienda online.

Funciones
---------
- Gestión de Categorías
- Gestión de Proveedores
- Gestión de Productos
- Gestión de Clientes
- Gestión de Pedidos
- Reportes
- Gráficas estadísticas

Tecnologías
-----------
- Python
- Flask
- Flask-AppBuilder
- MySQL
- Bootstrap
